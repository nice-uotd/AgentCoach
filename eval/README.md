# 评测说明

## BM25 vs Hybrid 检索选题相关性评测

### 目标
对比纯 BM25 关键词检索与 Hybrid 检索（BM25 + 向量语义 + RRF 融合）在面试题匹配场景下的效果差异。

### 测试集
- **10 份模拟简历**：覆盖纯 RAG、纯 Agent、全栈、转行、科研等不同背景
- **3 个岗位方向**：AI Agent 实习生、RAG 工程师、LLM 应用开发
- **92 道题库**：7 个类别

### 评估指标
- **选题相关性**：检索出的 top-5 题目与（简历+JD）的相关程度（1-5分）
- 使用标签重叠度作为启发式评分（无需 LLM API）

### 运行方式

```bash
cd AgentCoach
python -m eval.run_eval
```

结果保存在 `eval/eval_results.json`。

### 预期结论
Hybrid 检索利用向量语义匹配弥补 BM25 在同义词和语义相似场景下的不足，提升选题相关性。
